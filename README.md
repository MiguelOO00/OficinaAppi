Codigo necessario no terminal para o servidor funcionar corretamente:

Inicializar e instalar dependencias:
npm init -y ----> para iniciar  o projeto node.js;
npm install express mongoose dotenv bcryptjs jsonwebtoken cookie-parser ----> para instalar todas as dependencias necessarias para os servidor;
(servidor backend, ligação ao MongoDB, variáveis do ambiente, encriptação das passwords,utenticação JWT(por token) e leitura de cookies);


Instalacao:
git clone <url-do-repositorio>
cd ProjetoFinalOficinaApp

npm install


Executar Projeto: 
node index.js;
